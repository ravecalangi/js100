# Stopwatch App

## Description

A simple Stopwatch application built with React that allows users to start, pause, and reset a timer. The timer updates every second in real time using React Hooks and state management.

## Features

* Start the stopwatch
* Pause the stopwatch
* Reset the timer to zero
* Real-time second counter
* Automatic singular and plural time formatting ("sec" and "secs")
* Built with React Hooks (`useState` and `useEffect`)

## Technologies Used

* JavaScript (ES6)
* React
* ReactDOM
* HTML
* Babel

## How to Run

1. Save the code in an `index.html` file.
2. Ensure you have an internet connection because React, ReactDOM, and Babel are loaded through CDN links.
3. Open `index.html` in any modern web browser.
4. The Stopwatch App will load automatically.

## Usage

1. Click **Start** to begin counting time.
2. Click **Pause** to stop the timer temporarily.
3. Click **Reset** to return the timer to 0.
4. The timer display updates every second while running.

### Example

**Action:**
Click **Start** and wait for 5 seconds.

**Output:**
Time: 5 secs

**Action:**
Click **Pause**

**Output:**
Timer stops counting and remains at the current value.

**Action:**
Click **Reset**

**Output:**
Time: 0
