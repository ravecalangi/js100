# Expense Tracker

## Description

A simple Expense Tracker application built with React that allows users to add, edit, and delete expenses. The app manages expense records dynamically using React state and updates the interface instantly without reloading the page.

## Features

* Add new expenses with a description and amount
* Edit existing expense entries
* Delete expenses from the list
* Real-time UI updates using React state management
* Input validation to prevent empty entries
* Dynamic button text for adding and editing expenses

## Technologies Used

* JavaScript (ES6)
* React
* ReactDOM
* HTML
* Babel

## How to Run

1. Download or copy the project code into an `index.html` file.
2. Make sure you have an internet connection since React, ReactDOM, and Babel are loaded through CDN links.
3. Open `index.html` in your web browser.
4. The Expense Tracker application will load automatically.

## Usage

1. Enter an expense description in the **Description** field.
2. Enter the expense amount in the **Amount** field.
3. Click **Add Expense** to save the expense.
4. Click **Edit** beside an expense to modify its details.
5. Update the values and click **Edit Expense** to save changes.
6. Click **Delete** to remove an expense from the list.

### Example

**Input:**
Description: Groceries

Amount: 500

**Output:**
Groceries 500
