# Todo List App

## Description

A simple Todo List application built with React that allows users to add, edit, and delete tasks. The application updates the task list dynamically using React state management without requiring a page refresh.

## Features

* Add new tasks to the list
* Edit existing tasks
* Delete tasks
* Real-time updates using React state
* Preloaded sample tasks
* Input validation to prevent empty tasks
* Dynamic button text for adding and editing tasks

## Technologies Used

* JavaScript (ES6)
* React
* ReactDOM
* HTML
* Babel

## How to Run

1. Save the code in an `index.html` file.
2. Ensure you have an internet connection because React, ReactDOM, and Babel are loaded through CDN links.
3. Open `index.html` in any modern web browser.
4. The Todo List App will launch automatically.

## Usage

1. Type a task in the input field.
2. Click **Add** to add the task to the list.
3. Click **Edit** beside a task to modify it.
4. Update the task text and click **Edit** to save changes.
5. Click **Delete** to remove a task from the list.

### Example

**Initial Tasks:**

* banana
* apple
* orange

**Input:**
Buy groceries

**Output:**

* banana
* apple
* orange
* Buy groceries

**Edit Example:**

Before:

* Buy groceries

Input:
Buy groceries and drinks

After:
