# Task Manager

## Description

A simple Task Manager application built with React that allows users to create, edit, complete, and delete tasks. The application uses React state management to update tasks dynamically without requiring a page refresh.

## Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed using checkboxes
* Completed tasks are displayed with a strikethrough effect
* Real-time updates using React state
* Prevents adding empty tasks

## Technologies Used

* JavaScript (ES6)
* React
* ReactDOM
* HTML
* Babel

## How to Run

1. Save the code in an `index.html` file.
2. Ensure you have an internet connection because React, ReactDOM, and Babel are loaded through CDN links.
3. Open `index.html` in any modern web browser.
4. The Task Manager application will launch automatically.

## Usage

1. Enter a task in the input field.
2. Click **Add** to create a new task.
3. Click the checkbox to mark a task as completed or incomplete.
4. Click **Edit** to modify an existing task.
5. Update the task text and click **Edit** to save changes.
6. Click **Delete** to remove a task from the list.
